## 版本元数据导出

> generated by Codex

用于导出客户端的默认配置，以供客户端导出/导入功能使用

```powershell
uv run python tools/generate_partial_export_metadata.py
```

```powershell
uv run python tools/generate_partial_export_metadata.py \
--source "D:\\2ds-client\versions\3-3-5" \
--output "D:\\metadata_3-3-5.zip"
```

目前导出的 ZIP 内容（2026-02-24）：

- `options.txt`（原始文件副本）
- `hashes/mods.sha1`（每行一个去重后的 SHA-1）
- `hashes/resourcepacks.sha1`（每行一个去重后的 SHA-1）
- `hashes/shaderpacks.sha1`（每行一个去重后的 SHA-1）
